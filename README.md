# GPA Tool（GPA工具）

THU查看总GPA和中文成绩单的爬虫

## 功能

* 查看总GPA
* 查看中文成绩单

## 还需要拓展的功能

* 保存所有历史GPA
* 通过总GPA的变化计算出未公开成绩科目的绩点
* GPA计算器

## 原理

分析`info.tsinghua.edu.cn`的各种跳转与授权，写出了一个爬虫

**因此只有在能登录info的情况下该app才能使用**
